# Login Page Project

This project is a simple frontend login page with a modern split-screen design. It includes a username/email field, password field, password visibility toggle, remember-me checkbox, and a sign-in button with basic form validation.

## Features
- Clean and responsive layout
- Left side welcome panel with background image
- Right side login form
- Password show/hide toggle using Font Awesome icons
- Alert messages for empty fields and successful login

## Project Structure
- `login/LOGIN.HTML` — main HTML structure for the login page
- `login/style.css` — styling for the layout, form, and buttons
- `login/app.js` — JavaScript for form validation and password toggle behavior

## How to Run
1. Open the project folder in your browser.
2. Open [login/LOGIN.HTML](login/LOGIN.HTML) directly, or use a live server extension for easier preview.
3. Make sure the image files used in the CSS are available in the same folder.

## Notes
- This is a frontend-only demo, so the login button does not connect to a real backend.
- The password field toggle changes the input type between password and text.
